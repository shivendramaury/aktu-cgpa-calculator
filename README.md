AKTU CGPA Calculator is a simple and user-friendly web application designed to help students of Dr. A.P.J. Abdul Kalam Technical University (AKTU) calculate their CGPA quickly and accurately. Users can enter their semester SGPA values, and the application instantly computes the overall CGPA. The project focuses on clean design, ease of use, and fast results, making it helpful for students to track their academic performance efficiently. 👉 https://aktu-sgpa-calculator.netlify.app/
